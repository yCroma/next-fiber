# 3D アニメーション Web ビューアーのバックエンド

フレームワークとして[Nest.js](https://docs.nestjs.com/)を利用します。

## presetting

nest.js の cli をダウンロードすることを忘れずに

```bash
$ npm i -g @nestjs/cli
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
